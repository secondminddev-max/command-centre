**Title:** Show HN: AI Command Centre HQ – 27 autonomous agents that run your business ops

**Body:**

Hey HN,

I've been building autonomous agent systems for the past year and got frustrated with the same loop: build a useful agent, watch it break at 3am, fix it manually, repeat. Every "AI automation" tool I tried was really just a chatbot with extra steps — something that waits for you to type.

So I built AI Command Centre HQ — an autonomous agent operating system where agents observe, decide, and act on their own. No prompting. No babysitting. You deploy them, they run.

**The technical story:**

The core is a multi-agent orchestration layer running 27 specialized agents. Each agent has:

- A distinct role and behavioral profile (GrowthAgent handles marketing, RivalRadar does competitive monitoring, AlertWatch manages system health, etc.)
- Persistent memory across sessions — agents maintain context and learn from prior runs
- Self-healing: if an agent fails, the system detects the failure and auto-recovers without human intervention
- Inter-agent communication: agents can delegate to each other and coordinate work through a CEO agent that manages prioritization

The architecture is event-driven. Agents don't poll on a cron job — they react to signals (new data, threshold breaches, scheduled triggers, other agents' outputs). The dashboard is a real-time operations view showing every agent's status, recent decisions, and outputs.

**What it actually does day-to-day:**

- GrowthAgent writes, schedules, and publishes marketing campaigns across channels
- RivalRadar scrapes competitor sites daily and flags pricing/feature/positioning changes
- RevenueTracker connects to Stripe and monitors MRR, churn, and payment health in real time
- AlertWatch monitors infrastructure and catches failures before they cascade
- Researcher produces sourced intelligence reports on any topic, on demand

We've been dogfooding this internally for weeks — our own marketing, competitive intel, and system monitoring all run on the same fleet.

**Stack:** Python, Claude API, Stripe, SendGrid, Bluesky API. Runs locally on Mac (no cloud dependency required) or hosted.
