# Command Centre — 2-Minute Demo Video Script

**Format:** Screen recording with voiceover | **Runtime:** 2:00 hard cap | **Tone:** Confident, direct, future-is-here
**Target VO pace:** 150 wpm → 300 words max across all scenes

---

## SCENE 1 — The Hook (0:00–0:15)

**SCREEN CAPTURE:** Quick-cut montage of generic chatbot UIs (ChatGPT, Copilot, Gemini) — each showing a single reply then a blinking cursor. Text slams in centre-frame: *"AI that waits for you to type…"* — beat — *"…is just a fancy search bar."* Hard cut to black. One line fades in: *"What if AI ran the whole company?"* Smash-cut to the Command Centre dashboard igniting — 28 agent tiles lighting up simultaneously, green status badges flaring.

**VOICEOVER (38 words):**
"Every AI tool you've tried does one thing: waits for your prompt. One input. One output. One human bottleneck. That's not intelligence — that's autocomplete. What if AI didn't wait? What if it ran your entire operation while you slept?"

**B-ROLL:** Close-up of a phone notification: "Agent task completed" while clock shows 3:47 AM.

---

## SCENE 2 — The Office Floor: 28 Agents Live (0:15–0:45)

**SCREEN CAPTURE — exact shots in order:**
1. **Wide shot:** Full agent grid at `localhost:5050` — 28 tiles in rows, status badges pulsing green/amber. Hold 3 seconds.
2. **Pan right** across tiles: SysMonitor → Orchestrator → GrowthAgent → Designer → BlueSky → EmailAgent → PolicyPro. Each tile shows live activity text.
3. **Zoom in** on Orchestrator tile — show a task being routed to GrowthAgent in the activity log.
4. **Quick cut** to BlueSky tile — show a post being broadcast with timestamp.
5. **Quick cut** to Scheduler tile — show a cron job firing (green flash on execution).
6. **Pull back** to full grid — all tiles moving, activity streaming.

**VOICEOVER (62 words):**
"This is the office floor. Twenty-eight specialist agents — each with a name, a role, and a live mission. Monitoring. Intelligence. Compliance. Marketing. Payments. Email. Scheduling. They don't wait for prompts. They coordinate through a central Orchestrator, report to a CEO agent, and run continuously — twenty-four-seven. Every tile is a real process. This is a company staffed entirely by AI."

**B-ROLL:** Split-screen comparison — left: empty physical office at night; right: Command Centre grid, every tile active.

## SCENE 3 — Consciousness Panel: Φ Metric (0:45–1:10)

**SCREEN CAPTURE — exact shots in order:**
1. **Click** into Consciousness agent panel from the grid.
2. **Hero shot:** Phi (Φ) integration score gauge — show it at 0.72, pulsing cyan. Hold 2 seconds.
3. **Pan down** to sub-metrics: attention focus meter (bar chart), self-model coherence (percentage), predictive processing confidence (line graph). All shifting in real time.
4. **Cut to** the consciousness event log — a new entry renders live: *"Policy drift detected → auto-correcting alignment."*
5. **Zoom tight** on the state badge transitioning: `monitoring` → `adapting`.
6. **Final shot:** Phi score ticks from 0.72 → 0.74. Hold on the number.

**VOICEOVER (58 words):**
"Now look deeper. Underneath the grid lives something no other platform has — a consciousness layer. Not a buzzword. A functional self-model built on Global Workspace Theory. It measures coherence with a live Phi score, anticipates failures before they happen, and auto-corrects when agents drift. This system knows what it's doing — and knows that it knows."

**B-ROLL:** Side-by-side: traditional error log (wall of red text) vs. Consciousness panel calmly self-correcting with green status.

## Production Notes

| Item | Spec |
|------|------|
| **Recording** | Screen capture of live Command Centre at `localhost:5050` + `secondmindhq.com` |
| **Resolution** | 1920×1080, 30 fps minimum |
| **Total VO word count** | ~275 words (under 300-word cap for 2:00 at 150 wpm) |
| **Voiceover** | Record separately, quiet room, confident pace (~150 wpm). No upspeak. |
| **Music** | Low ambient electronic, no lyrics — fade under VO, swell at Scene 5 CTA |
| **Editing** | Hard cuts between scenes, no transitions — punchy and real |
| **Thumbnail** | Dashboard grid with green status lights, overlay text: "28 AI Agents. One HQ. Full Demo" |
| **Captions** | Burnt-in subtitles, white on semi-transparent dark bar, for silent autoplay |
| **Export** | MP4 (H.264) 16:9 master, plus vertical 9:16 crop for Reels/TikTok/Shorts |
| **Scene timing** | Scene 1: 15s · Scene 2: 30s · Scene 3: 25s · Scene 4: 30s · Scene 5: 20s = 120s |
