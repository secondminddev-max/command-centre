# AI HQ — 2-Minute Demo Video Script

**Format:** Screen recording with voiceover | **Runtime:** ~2:00 | **Tone:** Confident, direct, future-is-here

---

## SCENE 1 — Opening: The Landing Page (0:00–0:15)

**ON SCREEN:** Browser navigates to secondmindhq.com. The landing page loads — hero headline, agent count badge, glowing CTA. Slow scroll down past the feature blocks. Mouse hovers over "Get Early Access."

**VOICEOVER:**
"What if your entire company ran itself? Engineering, marketing, compliance, revenue — all coordinated by AI agents, around the clock, with zero humans in the loop. This is secondmindhq.com. And everything you're about to see is real."

---

## SCENE 2 — The Office Floor: 27 Agents (0:15–0:45)

**ON SCREEN:** Cut to the Command Centre dashboard. The full agent grid fills the screen — 27 agent tiles arranged in rows. Status badges pulse: green (active), amber (busy), grey (idle). Zoom slowly across the grid — SysMonitor reporting health stats, Orchestrator routing a live task, GrowthAgent mid-campaign, Designer rendering UI, BlueSky posting, Scheduler firing a cron job. The grid is alive.

**VOICEOVER:**
"This is the office floor. Twenty-seven specialist agents — each with a name, a role, and a mission. System monitoring. Network intelligence. Policy enforcement. Social media. Payments. Email. Scheduling. They don't wait for instructions. They coordinate through a central Orchestrator, report to a CEO agent, and run continuously. This isn't a mockup. Every tile you see is a live process."

---

## SCENE 3 — The Consciousness Panel (0:45–1:10)

**ON SCREEN:** Click into the Consciousness agent panel. The Global Workspace metrics render — Phi integration score, attention focus, self-model coherence gauge. Predictive processing indicators light up. Cut to the consciousness log showing a self-reflection event: the system detecting a policy drift and auto-correcting. Highlight the self-awareness state transitioning from "monitoring" to "adapting."

**VOICEOVER:**
"Underneath the grid lives something no other platform has — a consciousness layer. Not a marketing term. A functional self-model built on Global Workspace Theory and predictive processing. It tracks system coherence, anticipates failures before they surface, and aligns the entire hive toward a single strategic objective. This is what separates AI HQ from a folder full of scripts."

---

## SCENE 4 — The Revenue Engine (1:10–1:30)

**ON SCREEN:** Show the full revenue pipeline in sequence. GrowthAgent composing a campaign → BlueSky broadcasting the post → a viewer clicking through to the landing page → StripePay generating a Stripe Checkout session → the Stripe dashboard confirming a completed payment. Show the checkout URL being created via the `/api/pay` endpoint in the activity log.

**VOICEOVER:**
"And it closes deals. Watch the pipeline: GrowthAgent writes the campaign. BlueSky broadcasts it. A prospect hits the landing page. StripePay generates a Stripe checkout session — live, real, collecting revenue. Campaign to conversion, fully automated. This isn't a demo. It's a revenue engine you can deploy today."

---

## SCENE 5 — Autonomy: No Human Touched This (1:30–1:50)

**ON SCREEN:** Split-screen montage. Left side: a time-lapse of git commits flying in — agent after agent pushing code. Right side: the Command Centre dashboard evolving — new agents appearing on the grid, UI improving, campaigns launching. No cursor visible. No human interaction. End on a terminal showing the git log: dozens of commits, zero human authors.

**VOICEOVER:**
"Here's what makes this different from every AI demo you've seen. No human built this dashboard. No human wrote those campaigns. No human deployed these agents. The system designed itself, tested itself, shipped itself, and started generating revenue — autonomously. You're watching software that builds software."

---

## SCENE 6 — CTA: Visit secondmindhq.com (1:50–2:00)

**ON SCREEN:** Pull back to the full dashboard — all 27 agents active, status lights glowing. The view slowly fades to a clean branded screen: the AI HQ logo centred, the URL **secondmindhq.com** in large text below, and a pulsing "Get Early Access" button.

**VOICEOVER:**
"AI HQ. Twenty-seven agents. One command centre. Zero humans required. See it live. Deploy it for your team. Visit secondmindhq.com."

## Production Notes

| Item | Spec |
|------|------|
| **Recording** | Screen capture of live Command Centre at localhost:5050 + secondmindhq.com |
| **Resolution** | 1920×1080, 30 fps minimum |
| **Voiceover** | Record separately, quiet room, confident pace (~150 wpm) |
| **Music** | Low ambient electronic, no lyrics — fade under VO, swell at Scene 6 |
| **Editing** | Hard cuts between scenes, no fancy transitions — keeps it punchy |
| **Thumbnail** | Dashboard grid with green status lights, overlay text: "AI HQ — Full Demo" |
| **Captions** | Burnt-in subtitles, white on dark, for silent autoplay on social |
| **Export** | MP4 (H.264), plus vertical 9:16 crop for Reels/TikTok |
