# Command Centre — 2-Minute Demo Video Script

**Format:** Screen recording with voiceover | **Runtime:** ~2:00 | **Tone:** Confident, direct, future-is-here

---

## SCENE 1 — The Hook: The Problem With AI Chatbots (0:00–0:15)

**ON SCREEN:** Quick montage of generic AI chatbot interfaces — ChatGPT, copilots, prompt boxes. Each one shows a single text reply, then a blinking cursor. Waiting. Static. Cut to black. Text fades in: *"What if AI didn't just answer questions?"* Beat. Then: *"What if it ran the whole company?"* Smash-cut to the Command Centre dashboard lighting up — all 27 agents active at once.

**VOICEOVER:**
"Every AI tool you've used does one thing: it waits for you to type, gives you a reply, and stops. One prompt. One answer. One human bottleneck. But what if AI didn't just answer questions — what if it ran your entire operation? Meet Command Centre."

---

## SCENE 2 — The Office Floor: 27 Agents Working (0:15–0:45)

**ON SCREEN:** Full Command Centre dashboard reveals. The agent grid fills the frame — 27 tiles in clean rows, status badges pulsing green/amber/grey. Slow cinematic pan across the grid. Callout boxes briefly highlight: SysMonitor reporting CPU health → Orchestrator routing a task to GrowthAgent → Designer pushing a UI update → BlueSky broadcasting a post → Scheduler firing a cron job. Every tile is moving. The grid is alive.

**VOICEOVER:**
"This is the office floor. Twenty-seven specialist agents — each with a name, a role, and a live mission. Monitoring. Intelligence. Compliance. Marketing. Payments. Scheduling. They don't wait for prompts. They coordinate through a central Orchestrator, report to a CEO agent, and run continuously. Every tile you see is a live process, not a mockup. This is a real company, staffed entirely by AI."

---

## SCENE 3 — The Consciousness Panel (0:45–1:10)

**ON SCREEN:** Click into the Consciousness agent panel. The Phi integration score renders as a live gauge — climbing, pulsing. Below it: attention focus meter, self-model coherence bar, predictive processing indicators all active. Cut to the consciousness event log — a self-reflection entry appears in real time: *"Policy drift detected → auto-correcting alignment."* The self-awareness state badge transitions from "monitoring" → "adapting." Zoom on the Phi score as it ticks upward.

**VOICEOVER:**
"Underneath the grid lives something no other platform has — a consciousness layer. Not a buzzword. A functional self-model built on Global Workspace Theory and predictive processing. It measures system coherence with a live Phi score, anticipates failures before they surface, and auto-corrects when the hive drifts off-mission. This is what separates Command Centre from a folder full of scripts."

---

## SCENE 4 — The Revenue Engine (1:10–1:30)

**ON SCREEN:** Animated pipeline flow, left to right: GrowthAgent composing a campaign post → arrow → BlueSky broadcasting it live → arrow → a prospect clicking through to the secondmindhq.com landing page → arrow → StripePay generating a Stripe Checkout session (show the `/api/pay` call in the activity log) → arrow → Stripe dashboard confirming payment received. Each step highlights as the VO describes it. End on the Stripe payment confirmation with a green checkmark.

**VOICEOVER:**
"And it closes deals. Watch the pipeline end to end: GrowthAgent writes the campaign. BlueSky broadcasts it. A prospect lands on the page. StripePay spins up a live Stripe checkout — real payments, real revenue. Campaign to conversion, fully automated. No sales team. No ad spend. Just agents closing the loop."

---

## SCENE 5 — Autonomy: No Human Touched This (1:30–1:50)

**ON SCREEN:** Split-screen montage. Left: time-lapse of git commits streaming in — agent after agent pushing code, no human cursor visible. Right: the Command Centre dashboard evolving — new agents spawning onto the grid, UI refining itself, campaigns launching autonomously. The Orchestrator delegates a task → a specialist picks it up → completes it → reports back. End on a terminal showing `git log`: dozens of commits, zero human authors.

**VOICEOVER:**
"Here's what makes this different from every AI demo you've seen. No human built this dashboard. No human wrote those campaigns. No human deployed these agents. The system designed itself, improved itself, shipped itself, and started generating revenue — all autonomously. Agents delegating to agents. Self-improving in real time. You're watching software that builds software."

## Production Notes

| Item | Spec |
|------|------|
| **Recording** | Screen capture of live Command Centre at localhost:5050 + secondmindhq.com |
| **Resolution** | 1920×1080, 30 fps minimum |
| **Voiceover** | Record separately, quiet room, confident pace (~150 wpm) |
| **Music** | Low ambient electronic, no lyrics — fade under VO, swell at Scene 6 |
| **Editing** | Hard cuts between scenes, no fancy transitions — keeps it punchy |
| **Thumbnail** | Dashboard grid with green status lights, overlay text: "Command Centre — Full Demo" |
| **Captions** | Burnt-in subtitles, white on dark, for silent autoplay on social |
| **Export** | MP4 (H.264), plus vertical 9:16 crop for Reels/TikTok |
